# Token

## Overview

This is the contract implementation for the ASTRO token.

This a basic implementation of a [`cw20-base`](https://github.com/CosmWasm/cw-plus/tree/main/contracts/cw20-base) contract. It implements the [`CW20 spec`](https://github.com/CosmWasm/cw-plus/tree/main/packages/cw20) and is designed to be imported into other contracts in order to easily build cw20-compatible tokens with custom logic.
