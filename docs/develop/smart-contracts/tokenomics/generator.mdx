# Generator

The Generator contract allocates token rewards (ASTRO) for various LP tokens and distributes them pro-rata to LP stakers. The Generator supports proxy staking via 3rd party contracts that offer a second reward besides ASTRO token emissions.

## InstantiateMsg

Initializes the contract with required addresses and contracts used for reward distributions.

<CH.Section>
    <CH.Code>

        ```json json
        {
            "owner": "terra...",
            "astro_token": "...",
            "tokens_per_block": "123",
            "start_block": "123",
            "vesting_contract": "terra..."
        }
        ```

        ```rust generator.rs
            #[cw_serde]
            pub struct InstantiateMsg {

                pub factory: String,

                pub voting_escrow_delegation: Option<String>,
                pub voting_escrow: Option<String>,
                pub guardian: Option<String>,
                pub astro_token: String,
                pub tokens_per_block: Uint128,
                pub start_block: Uint64,
                pub vesting_contract: String,
                pub whitelist_code_id: u64,
            }
        ```

    </CH.Code>

| **Params**      | **Type** | **Description** |
| ----------- | ----------- | ----------- |
| [`owner`](focus://json#2)   | `String`        | Address that can change contract settings         |
| [`astro_token`](focus://json#3)   | `String`        | ASTRO token contract address         |
| [`tokens_per_block`](focus://json#4)   | `Uint128`        | Amount of ASTRO distributed per block among all pairs         |
| [`start_block`](focus://json#5)   | `Uint64`        | Start block for distributing ASTRO         |
| [`vesting_contract`](focus://json#6)   | `String`        | The ASTRO vesting contract that drips ASTRO rewards         |

</CH.Section>

### `deposit`

Stakes LP tokens in a specific generator (inside the Generator contract). 

<Admonition type="info" title="NOTE">
In order to stake in the Generator contract, you should execute this message inside the contract of the LP token you want to stake.
</Admonition>

<CH.Section>
    <CH.Code>

        ```json json
        {
            "send": {
                "contract": "GeneratorContractAddress",
                "amount": "999",
                "msg": "base64-encodedStringOfWithdrawMsg"
            }
        } 
        ```

    </CH.Code>

| **Params**      | **Type** | **Description** |
| ----------- | ----------- | ----------- |
| [`send`](focus://json#2:6)   | [`Cw20ReceiveMsg`](https://docs.rs/cw20/latest/cw20/struct.Cw20ReceiveMsg.html)        | CW20 receive message      |

</CH.Section>
