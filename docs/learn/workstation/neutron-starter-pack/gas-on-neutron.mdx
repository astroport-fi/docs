# Gas on Neutron

Neutron  supports both ATOM and NTRN as gas tokens for executing transactions. The recommended method to obtain gas on Neutron is to bridge ATOM from a centralized exchange to the Cosmos Hub network. From here, ATOM can be bridged using the IBC network to Neutron.

**This tutorial assumes you were able to obtain ATOM from an exchange (Coinbase, Binance, KuCoin, Kraken, etc.) and were able to send it to your Cosmos wallet.** Lets continue below.

### Step 1: Head over to [tfm.com/bridge](https://tfm.com/bridge)

Once we have ATOM in our Cosmos wallet, we can transfer our ATOM from the Cosmos Hub to Neutron using IBC. To do this, we can use the [tfm.com/bridge](https://tfm.com/bridge) UI. Once you're in the page, select **"Connect Wallet"** to continue. 

<img src={Gas1Url} style={{ paddingBottom: 10 }} />

### Step 2: Connect Cosmos Hub Wallet

From the dropdown menus, select the Cosmos Hub netwrok and a supported wallet. 

<img src={Gas2Url} style={{ paddingBottom: 10 }} />

Once connected, your wallet for Cosmos Hub will appear in the wallet drop down menu.

<img src={Gas2BUrl} style={{ paddingBottom: 10 }} />

### Step 3: Connect Neutron wallet 

Now that we are connected to the Cosmos Hub network, we will want to do the same for our Neutron wallet. Select the add wallet icon and add the Neutron network and a wallet. 

<img src={Gas3Url} style={{ paddingBottom: 10 }} />

Now you should have two connected wallets with TFM. 

<img src={Gas3BUrl} style={{ paddingBottom: 10 }} />

### Step 4: Select Transfer Route

Now we are ready to begin our transfer. Select the transfer route (Cosmos Hub to Neutron). 

<img src={Gas4Url} style={{ paddingBottom: 10 }} />

### Step 5: Select Token and Amount 

Next, select the token (ATOM) and an amount to send. Select "Transfer" to continue.

<img src={Gas5Url} style={{ paddingBottom: 10 }} />

### Step 6: Confirm and Sign Transaction 

Lastly, confirm your transaction. 

<img src={Gas6Url} style={{ paddingBottom: 10 }} />

And sign using your wallet. 

<img src={Gas6BUrl} style={{ paddingBottom: 10 }} />

You will be prompted once the transaction is complete with a link to the transaction hash where you can get more details. 

<img src={Gas6CUrl} style={{ paddingBottom: 10 }} />
